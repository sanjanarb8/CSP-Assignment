# 🧠 CSP Assignment

## 📌 Overview

This project implements multiple **Constraint Satisfaction Problems (CSP)** using Python and backtracking techniques.

---

## 📂 Problems Implemented

### 1. Australia Map Coloring

* States: WA, NT, QLD, SA, NSW, V, T
* Objective: Assign colors so that no neighboring states share the same color

---

### 2. Telangana Map Coloring (33 Districts)

* All districts of Telangana considered
* Applied CSP constraints using adjacency relationships
* Ensures no adjacent districts have the same color

---

### 3. Sudoku Solver

* Solves a 9×9 Sudoku grid
* Uses backtracking algorithm
* Ensures valid row, column, and subgrid constraints

---

### 4. Cryptarithmetic Puzzle

* Problem:
  SEND + MORE = MONEY
* Each letter represents a unique digit
* Solved using permutations and constraint checking

---

## ⚙️ Concepts Used

* Constraint Satisfaction Problem (CSP)
* Backtracking Algorithm
* Variables, Domains, Constraints
* Python Data Structures
